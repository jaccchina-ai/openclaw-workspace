## 1. 项目概述

### 1.1 背景
T01目前只有实时选股功能，缺乏历史回测验证。需要建立自动化回测框架，用历史数据验证策略有效性。

### 1.2 目标
- 创建自动化回测引擎，模拟完整交易流程
- 计算关键绩效指标（胜率、盈亏比、最大回撤、夏普比率）
- 每周自动生成回测报告
- 对比回测与实盘表现，发现策略失效信号

### 1.3 成功标准
- 能够回测过去1年的涨停股数据
- 回测报告生成时间 < 5分钟
- 策略失效预警准确率 > 80%

## 2. 系统架构

### 2.1 核心组件

```
┌─────────────────────────────────────────────────────────────┐
│                    T01 Backtest Engine                      │
├─────────────────────────────────────────────────────────────┤
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐      │
│  │ Data Loader  │→ │ Simulator    │→ │ Metrics      │      │
│  │ (历史数据)   │  │ (交易模拟)   │  │ (绩效计算)   │      │
│  └──────────────┘  └──────────────┘  └──────────────┘      │
│         ↓                 ↓                 ↓               │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐      │
│  │ Tushare API  │  │ Strategy V2  │  │ Report Gen   │      │
│  │ (limit_list) │  │ (评分逻辑)   │  │ (报告生成)   │      │
│  └──────────────┘  └──────────────┘  └──────────────┘      │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 数据流

1. **数据加载**: 从Tushare获取历史涨停股数据
2. **策略模拟**: 对每一天应用T01评分逻辑
3. **交易模拟**: 模拟T+1日竞价买入，T+2日收盘卖出
4. **绩效计算**: 计算胜率、收益率、回撤等指标
5. **报告生成**: 生成可视化回测报告

## 3. 技术方案

### 3.1 方案对比

| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| A. 独立回测引擎 | 完全控制，灵活 | 工作量大 | ⭐⭐⭐⭐ |
| B. 复用Strategy V2 | 代码复用高 | 需要适配 | ⭐⭐⭐⭐⭐ |
| C. 使用Backtrader | 功能完善 | 学习成本 | ⭐⭐⭐ |

**推荐**: 方案B - 复用现有Strategy V2，在其基础上增加回测功能

### 3.2 核心设计决策

1. **数据范围**: 回测过去1年（约250个交易日）
2. **模拟逻辑**: 
   - T日20:00选股评分
   - T+1日09:25开盘价买入
   - T+2日15:00收盘价卖出
3. **滑点考虑**: 买入价格 = 开盘价 × 1.005 (0.5%滑点)
4. **手续费**: 单边0.03%，双边0.06%

## 6. 配置扩展

在`config.yaml`中增加回测配置：

```yaml
backtest:
  enabled: true
  lookback_days: 252  # 回测1年
  initial_capital: 1000000  # 初始资金100万
  commission_rate: 0.0003  # 手续费率0.03%
  slippage: 0.005  # 滑点0.5%
  position_size: 0.2  # 单票仓位20%
  max_positions: 3  # 最大持仓3只
  report_schedule: "weekly"  # 每周生成报告
  report_time: "06:00"  # 北京时间周一早上6点
  alerts:
    backtest_vs_live_deviation: 0.20  # 回测vs实盘偏离20%预警
```
