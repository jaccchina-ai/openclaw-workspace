### 问题发现
**心跳检查时发现**: T99扫描连续第7天失败，但错误类型变化
- **错误**: `TypeError: Object of type bool is not JSON serializable`
- **位置**: `main.py` 第18行 `_print` 函数中的 `json.dumps()`
- **根本原因**: `short_term_signal_engine` 返回的数据中包含 `numpy.bool_` 类型
- **影响**: 即使扫描成功，也无法输出JSON结果，导致整体失败

### 修复实施
**时间**: 21:30 UTC
**方案**: 添加自定义JSON编码器处理numpy类型
**修改文件**: `/root/.openclaw/workspace/skills/a-share-short-decision/main.py`

**具体修改**:
1. 添加numpy导入: `import numpy as np`
2. 修改`_print`函数，添加`EnhancedJSONEncoder`类处理:
   - `np.bool_` → `bool`
   - `np.integer` → `int` 
   - `np.floating` → `float`
   - `np.ndarray` → `list`

**验证测试**:
```python
# 测试numpy bool序列化 - ✅ 成功
data = {'field1': np.bool_(True), 'field2': np.bool_(False)}
json.dumps(data, cls=EnhancedJSONEncoder)  # 正常序列化
```
